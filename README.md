# Chat UI

Chat UI is a simple, modern chat interface inspired by popular messaging applications.  
This project is created using **HTML and Tailwind CSS**.

## Features

- Custom app name and logo text
- Contact list with chat preview
- Typing status and message time
- Clean and modern UI
- Responsive design

## Technologies Used

- HTML
- Tailwind CSS
